# 👨‍💼 HR Attrition Analysis Dashboard

🚀 **End-to-End Data Analyst Project | Python + SQL + Power BI**

---

## 📊 Project Overview

This project analyzes **employee attrition data** to identify patterns, trends, and key factors affecting employee turnover.

An interactive dashboard is built using **Power BI** to help HR teams make **data-driven decisions**.

---

## 🎯 Business Problem

Organizations face challenges such as:

* High employee turnover
* Difficulty identifying attrition causes
* Lack of workforce insights
* Poor employee retention strategies

👉 This project helps in understanding and reducing attrition.

---

## 🛠️ Tech Stack

* **Python** → Data Cleaning & Preprocessing (Pandas, NumPy)
* **SQL (PostgreSQL)** → Data Transformation & Querying
* **Power BI** → Dashboard & Visualization
* **Excel / CSV** → Dataset

* ❌ Attrition Count: **237**

---

### 🔹 Attrition Analysis

#### 📌 By Job Role

* Highest attrition in **Laboratory Technician & Sales roles**

#### 📌 By Department

* Research & Development shows highest attrition

#### 📌 By OverTime

* Employees with overtime show higher attrition

#### 📌 By Tenure

* Employees with **0–2 years experience** leave more frequently

#### 📌 Work-Life Balance

* Lower work-life balance → higher attrition

---

## 🔍 Key Insights

* ⚠️ Early-stage employees are more likely to leave
* ⏱️ Overtime significantly impacts attrition
* 🏢 Certain departments have higher turnover
* 😊 Work-life balance plays a critical role
